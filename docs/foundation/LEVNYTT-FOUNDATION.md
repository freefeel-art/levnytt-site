# LEVNYTT FOUNDATION

## Purpose

LevNytt is a Swedish informational website focused on:

- NeoLife
- Golden Home Care
- science-based information
- consumer education
- sustainability
- long-term trust

The site prioritizes:

1. Clarity
2. Trust
3. Education
4. User value

---

## Design System

### Colors

Official color system:

- --gold
- --gold-light
- --gold-dark

- --green
- --green-light
- --green-dark

- --cream
- --cream-dark

- --white

- --ink
- --ink-muted

- --border

### Typography

Headings:
- Playfair Display

Body:
- Inter

### Visual Style

Use:

- generous spacing
- readable content width
- rounded corners
- soft shadows
- clean layouts

Avoid:

- aggressive marketing design
- countdown timers
- hype elements
- dark-pattern UX

---

## CTA Architecture

Preferred destinations:

- Webshop
- Registrering
- Golden Home Care
- Historia
- Affärsmöjlighet

CTA style should be informative and helpful.

Avoid:

- Buy Now
- Limited Time
- Guaranteed Results
- Easy Money
- Income Promises

---

## Builder Rule

Before creating a new page:

1. Follow the LevNytt design system.
2. Follow the existing navigation structure.
3. Use approved components.
4. Reuse existing patterns before creating new ones.
5. Match the style of existing LevNytt pages.

The page must feel like a natural part of LevNytt.
