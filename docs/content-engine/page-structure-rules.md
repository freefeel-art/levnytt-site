
# LevNytt Principles

- Fakta före hype
- Värde före pris
- Konsekvens före perfektion
- Långsiktighet före genvägar
- Förstå först, besluta sedan
- Nyfikenhet före tvärsäkerhet

These principles guide:
- article writing
- CTA design
- internal linking
- authority pages
- pull quotes
- product positioning

Pull quotes should reinforce a LevNytt Principle when appropriate, but are never required.

# Pull Quote Rules

Purpose:

Pull quotes highlight important ideas,
not famous people.

Rules:

- Optional, never required.
- Use 0-2 pull quotes per article.
- Pull quotes should reinforce a LevNytt Principle.
- Prefer original article insights over external quotes.
- Warren Buffett, Steve Jobs or other sources may inspire the idea,
  but the pull quote should normally be written in LevNytt's own voice.
- Pull quotes must be short, memorable and easy to scan.
- Do not force pull quotes into articles that do not naturally contain one.

Examples:

"Fakta före hype."

"Värde handlar om vad du får, inte bara vad du betalar."

"Konsekvens slår nästan alltid perfektion."

"Långsiktighet är ofta underskattad eftersom resultaten syns sent."

# Related Articles Rules

Use exactly 3 related articles:

- 1 same topic
- 1 supporting topic
- 1 next logical step

Build a reader journey, not a random link list.
