## VIKTIGT — Prislogik

- **Kundpris** = Pris inkl. moms (som det står i kundprislistan)
- **Återförsäljarpris** = Distributor-pris exkl. moms + moms tillagd = **Pris inkl. moms**
- **Moms kosttillskott & viktkontroll:** 6%
- **Moms rengöring, hår/kropp, hudvård:** 25%
- **Besparing SEK** = Kundpris − Återförsäljarpris (inkl. moms)
- **Besparing %** = Besparing SEK ÷ Kundpris × 100

Alla besparingar är beräknade — inget är påhittat.

## CALCULATOR VARIABLES

Dessa variabler används i LevNytt:s interaktiva verktyg.

### Besparingskalkylatorn (Savings Calculator)

```
CUSTOMER_PRICE        = Kundpris inkl. moms (SEK)
DISTRIBUTOR_PRICE     = Återförsäljarpris inkl. moms (SEK)
SAVINGS_SEK           = CUSTOMER_PRICE - DISTRIBUTOR_PRICE
SAVINGS_PCT           = (SAVINGS_SEK / CUSTOMER_PRICE) * 100
MONTHLY_SAVINGS       = SAVINGS_SEK * PURCHASES_PER_MONTH
ANNUAL_SAVINGS        = MONTHLY_SAVINGS * 12
```

### Kostnad Per Användning (Cost Per Use Calculator)

PRODUCT_PRICE         = Pris att beräkna på (kund eller återförs.)

COST_PER_USE          = PRODUCT_PRICE / USES_PER_PACK

### Affärsmöjlighet 2.0 — Inkomstpotential

```
CUSTOMER_PRICE        = Kundpris (vad kunden betalar)
DISTRIBUTOR_PRICE     = Återförsäljarpris (vad du betalar)
MARGIN_PER_UNIT       = CUSTOMER_PRICE - DISTRIBUTOR_PRICE
CUSTOMERS_COUNT       = Antal kunder
AVG_ORDER_VALUE       = Genomsnittligt ordervärde per kund
MONTHLY_REVENUE       = CUSTOMERS_COUNT * AVG_ORDER_VALUE
MONTHLY_MARGIN        = CUSTOMERS_COUNT * (AVG_ORDER_VALUE * MARGIN_PCT)
MARGIN_PCT            = ~0.21 (21% typisk marginal)

-- Exempelscenario (baserat på verkliga priser) --
10 kunder × 500:- snittorder = 5 000:- omsättning
Marginal (~21%): ~1 050:- per månad
Årsintäkt marginal: ~12 600:-
```
